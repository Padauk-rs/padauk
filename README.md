### ğŸ“– Introduction

**Padauk** is a cross-platform UI framework that allows you to write your application logic and UI layout entirely in **Rust**, while rendering **truly native components** on the device.

Unlike web-based frameworks or game engines, Padauk maps high-level Rust structs directly to:

* **Android**: Jetpack Compose (Material 3)

* **iOS**: UIKit / SwiftUI (Planned)

It features a robust **Dual-Layer Architecture**:

1. **Core SDK**: A pre-compiled native library handling the rendering engine.

2. **User App**: Your Rust code, compiled dynamically and linked via a high-performance FFI bridge.

### âœ¨ Features

* **Rust-First API**: Declarative UI syntax similar to SwiftUI or Jetpack Compose, but in Rust.

* **Native Rendering**: No WebViews, no canvas drawing. Uses platform-native widgets for accessibility and performance.

* **Type-Safe Assets**: The CLI automatically scans your assets/ folder and generates strict Rust constants (e.g., assets::ICONS_HOME).

* **Zero-Boilerplate Entry**: A simple attribute macro #[padauk::main] handles the platform handshake for you.

* **Smart CLI**: padauk run handles the entire toolchain: Rust compilation, FFI binding generation, asset syncing, and Gradle/Xcode builds.

### ğŸ›  Prerequisites

To develop with Padauk, you need:

1. **Rust Toolchain**: `rustup` installed.

2. **Android Development**:

   * Android Studio & SDK.

   * NDK (Side-by-side) installed via SDK Manager.

   * `rustup target add aarch64-linux-android` (and `x86_64` for emulators).

3. iOS Development (macOS only):

* Xcode.

* `rustup target add aarch64-apple-ios aarch64-apple-ios-sim`.

### ğŸ“¦ Installation

Since Padauk is currently in active development, it is best to build the CLI from source:
```
# 1. Clone the repository
git clone https://github.com/aungthuya-dev/Padauk.git
cd padauk

# 2. Build the Framework Assets (AAR/XCFramework)
cd padauk
chmod +x script/build-android-release.sh
./script/build-android-release.sh

# 3. Install the CLI binary.
cd padauk-cli
chmod +x script/build.sh
./script/build.sh
cargo install --path . --features embed-assets
```

### ğŸš€ Getting Started

**1. Create a new project**
```
padauk create my_awesome_app
cd my_awesome_app
```

**2. Write your UI (`rust/src/lib.rs`)**
```
use padauk::prelude::*;

// Auto-generated assets module
mod assets; 

#[padauk::main]
fn main() -> impl Widget {
    scaffold(
        column(vec![
            image(assets::LOGO_PNG)
                .width(120.0)
                .height(120.0)
                .fit(BoxFit::Contain),
            
            text("Welcome to Padauk")
                .size(24.0)
                .padding(16.0),
                
            text("Build native, stay rusty.")
                .color("#666666")
        ])
        .align(MainAxisAlignment::Center)
    )
    .app_bar(app_bar("My App"))
}
```

**3. Run on Android**

Ensure an emulator is running or a device is connected.
```
padauk run android
```

**4. Run on iOS (macOS only)**
```
padauk run ios
```
